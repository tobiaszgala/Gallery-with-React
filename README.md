# Simple Gallery App using React

Simple gallery app using React and Flickr API.

## Setup

Create config.js file in **src** directory containing flickr api key. 

```
// config.js structure
const key = '<insert key here>';
export default key;
```

## Usage 

```
// run 

npm install
npm start
```
